# 맛집지도 이미지 폴더

맛집 사진(대표 사진·메뉴판 등)을 이 폴더에 넣으세요.

## 사용법
1. 사진 파일을 이 `images/` 폴더에 복사 (예: `교대이층집_대표.jpg`)
2. `food-map.html` 의 해당 가게 데이터에 경로를 적습니다:

```js
photos: [
  'images/교대이층집_대표.jpg',
  { src: 'images/교대이층집_메뉴판.jpg', cap: '메뉴판' },
]
```

## 팁
- 파일명은 영문/숫자를 권장 (한글도 되지만 깨질 수 있어요)
- 사진이 너무 크면(5MB+) 로딩이 느려요. 1~2MB 이하로 줄여서 올리면 좋아요
- 외부 URL(`https://...jpg`)도 그대로 넣을 수 있지만, 사라지거나 막힐 수 있어 폴더 보관이 더 안전합니다
