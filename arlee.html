조율 요청마다 반복되던 BI 5개 수기 조회·정책 대입·매트릭스 계산을 LLM 에이전트 end-to-end 파이프라인으로 단독 설계·구현. 5단계(스캔 → 모드 분기 → 데이터 통합 → 룰 엔진 → 출력 분리)를 자동화해 판정 블록과 답변 초안을 동시 산출합니다.
